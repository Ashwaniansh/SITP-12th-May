Laptop Price Prediction System

About Project

This project is a Machine Learning based Laptop Price Prediction System. It predicts the expected price of a laptop based on its specifications like brand, processor, CPU, RAM, ROM, GPU, display size, operating system and warranty.

Project Features

  • Machine Learning based price prediction
  • Random Forest Regression model
  • User friendly prediction interface
  • Prediction history storage using SQLite database
  • Fast and accurate price estimation

Technology Used

  • Python
  • Flask Framework
  • Pandas & NumPy
  • Scikit-Learn
  • SQLite Database
  • HTML & CSS
